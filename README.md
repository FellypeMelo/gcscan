# GCScan

Analisador de conteúdo GC em sequências genômicas.

## Instalação

```bash
pip install -r requirements.txt
```

## Uso

1. Coloque seu arquivo FASTA na pasta `data/`
2. Execute: `python main.py`

## Funcionalidades

- Lê arquivos FASTA
- Calcula conteúdo GC usando Biopython
- Gera gráfico de barras com Matplotlib
