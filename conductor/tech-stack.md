# GCScan - Technology Stack

## Core Language
- **Python (3.7+):** Chosen for its extensive bioinformatics ecosystem and ease of integration with data science libraries.

## Bioinformatics Libraries
- **Biopython:** The industry standard for handling biological data formats like FASTA and providing utility functions for genomic calculations.

## Data Processing & Analysis
- **Pandas:** Used for structured data manipulation and statistical summaries.
- **Numpy:** Provides efficient numerical operations for large datasets.

## Visualization
- **Matplotlib:** Primary library for generating static charts and plots.
- **Altair:** Used for declarative, interactive statistical visualizations.

## Web Framework
- **Streamlit:** Facilitates the rapid creation of professional web-based bioinformatics dashboards directly from Python scripts.

## Quality Assurance & Testing
- **Pytest:** Mandatory framework for automated testing, ensuring calculation accuracy and application reliability through extensive unit and integration tests.

## Infrastructure & Architecture
- **Monolith Architecture:** The project is organized as a single repository containing core logic, a CLI entry point, and a web interface.
- **Git:** Version control system for collaborative development and tracking changes.
